Twin Cities Documentation

Maintenance of database and code

The application pulls data from a populated database of twinning cities, Edinburgh & Munich.

The application uses Edinburgh (Scotland) and Munich (Germany) for the location - these cities both have relatively large populations so they will have sufficient data to build our application from.

To help conceptualise how our data would interact and be stored in the database, we created a conceptual and logical ER model. Using Astah & mySQL to help render the models.
